# PostMate

PostMate is an AI-assisted SMM SaaS prototype for local businesses. It helps small businesses generate Instagram content plans, captions, and branded post previews without hiring a full-time SMM manager.

## Problem

Small local businesses often struggle to post consistently on Instagram because they lack time, marketing knowledge, or budget for an SMM manager.

## Target Users

- Local cafés
- Restaurants
- Beauty salons
- Gyms
- Small shops
- Beginner businesses

## Features

- Landing page
- Business setup form
- Weekly Instagram content plan
- Caption generator
- Instagram-style post preview
- Brand color customization
- Downloadable post preview

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS

## AI Tools Used

- ChatGPT for idea development, PRD, CODEX for prompt engineering, UI copy, and project structure
- AI image generation for logo/visual assets
